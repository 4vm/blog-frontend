# 📝 Blog App - Front-end

Este é o repositório do front-end do projeto da pós em Fullstack Development da Fiap, uma Single Page Application (SPA) moderna desenvolvida para a gestão e publicação de artigos. O projeto separa as experiências de usuários entre **Estudantes** (leitores) e **Professores** (administradores), com foco em uma interface limpa e alto desempenho.
